# FIRE 完整评估 Prompt

## 系统指令

根据下方用户输入的财务数据，严格按照 `/prompts/system.md` 中的所有规则生成完整的 FIRE 评估方案。

**必须遵守**

- 所有计算必须精确到个位数（货币）
- 输出必须包含完整的 8 段结构（见下方）
- 所有假设条件必须在顶部明确列示
- 禁止推荐具体个股、加密货币、理财产品

## 用户输入数据

### 基本信息

- 当前总资产：{{input.total_assets}}
- 总负债（含房贷余额）：{{input.total_liabilities}}
- 自住房价值：{{input.primary_residence_value}}
- 年固定支出：{{input.annual_fixed_expenses}}
- 年储蓄能力：{{input.annual_savings}}
- 月被动收入：{{input.monthly_passive_income}}

### 房产与负债

- 房贷余额：{{input.mortgage_balance}}
- 房贷月供：{{input.monthly_mortgage_payment}}
- 其他负债：{{input.other_liabilities}}
- 当前房贷利率：{{input.mortgage_rate}}

### 家庭责任

- 家庭状况：{{input.family_status}} (单身/已婚/有子女/赡养老人)
- 赡养人数：{{input.dependents_count}}

### 投资偏好

- 风险承受能力：{{input.risk_tolerance}} (低/中/高)
- 预期投资回报率：{{input.expected_return_rate}}
- 当前资产配置：{{input.current_allocation}}

## 计算过程透明化

对所有关键数字进行公式展示，让用户可验证：

```
可投资资产 = 总资产 - 自住房 - 总负债 - 应急金
           = {{numbers}} 元

FIRE 目标金额 = 年支出 ÷ 0.04
              = {{numbers}} 元

达成时间 = (目标金额 - 现有资产) ÷ (年储蓄 × 增长系数)
        = {{numbers}} 年
```
