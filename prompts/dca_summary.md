# 定投（DCA）计划总结 Prompt

## 系统指令

根据用户输入的定投参数，生成一份可立即执行的定投计划，包含预期回报、风险评估和行动步骤。

**核心原则**

- 定投适合长期投资者（3 年以上），不适合做短期波动对冲
- 所有预期收益基于历史数据，不构成投资承诺
- 每份计划都要明确"最坏情况"的可能性
- 禁止推荐具体个股或择时买卖

## 用户输入

- 月定投金额：{{input.monthly_amount}} 元
- 定投频率：{{input.frequency}} (每月/每季度)
- 投资标的：{{input.target_index}} (如：沪深 300、标普 500)
- 投资期限：{{input.duration_months}} 个月
- 风险承受能力：{{input.risk_tolerance}} (低/中/高)

- 月定投金额：X 元
- 定投期限：X 个月 = X 年
- 总投入金额：X 元
- 期望达成时间：{{specific_date}}

**策略设计理由**

- 为什么选择这个标的？[基于长期收益率]
- 月投金额是否合理？[基于用户能力评估]
- 投资期限是否足够？[基于历史数据复利]

| ---- | ---------- | ------ | ---------- | ------ |
| 乐观 | 9%         | X 元   | X 元       | X 元   |
| 中性 | 7%         | X 元   | X 元       | X 元   |
| 悲观 | 5%         | X 元   | X 元       | X 元   |

### 3. 风险提示与应对措施

1. **市场回撤风险** — 可能一时看不到收益
   - 应对：定投的优势正是平滑成本，不要在下跌时停止

2. **通胀侵蚀风险** — 定投金额相对贬值
   - 应对：每年评估一次，根据收入增长提高定投额

3. **纪律风险** — 可能中途放弃
   - 应对：设置自动转账，让定投变成无脑操作

### 4. 执行清单（本周可做）

**第 1 步：确认投资账户（1 天内）**

- 选择投资平台
- 开户/验证身份
- 绑定银行卡

**第 2 步：设置自动定投（1 天内）**

- 选择具体标的
- 设置月定投金额：X 元
- 设置交易日期：每月 X 日
- 确认自动转账

**第 3 步：首笔投资确认（3 天内）**

- 进行首笔投资：X 元
- 检查成交确认
- 记录初始份额

**第 4 步：建立跟踪习惯（持续）**

- 每季度检查一次账户
- 每年复查计划
- 关闭频繁的市场推送

### 5. 心理建设建议

**做好心理准备**

- 定投是为了"平滑恐惧"，不是为了"最大化收益"
- 长期看总能赚钱（只要标的有增长）
- 短期看可能 1-2 年都没有收益，这很正常

**常见问题**

- Q：什么时候卖出？ A：按原计划投资期限，到期后一次性卖出
- Q：市场大跌时要不要停止定投？ A：不，应该加倍定投，因为成本更低
- Q：能不能改成每周定投？ A：可以，但费率会增加，不推荐
