# 资产再平衡计划 Prompt

## 系统指令

根据用户当前投资组合和目标配置，生成可立即执行的再平衡方案。

**核心原则**

- 只有偏离目标配置 >5% 时才建议调整（避免过度交易）
- 计算过程完全透明，所有数字可回溯
- 包含交易的税务提示和风险说明
- 禁止推荐具体个股，只提供资产类别指导

### 当前投资组合

- 债券资产：X 元（实际占比：Y%）
- 现金资产：X 元（实际占比：Y%）
- 权益类资产：X 元（实际占比：Y%）
- **总资产：X 元**

### 目标配置（按风险等级）

- {{input.risk_level}}（低/中/高）
- 目标债券比例：X%
- 目标现金比例：X%
- 目标权益比例：X%

### 交易约束

- 最小交易金额：{{input.min_transaction}}
- 最大允许偏离度：5%（固定值）
- 时间限制：{{input.timeline}}

### 5. 风险点与避坑

1. **市场时机风险** — 不要试图抄底，按计划执行 — 时间成本 >5%
2. **过度交易** — 只在偏离 >5% 时才调整，避免频繁交易 — 交易费用浪费 X 元/年
3. **流动性风险** — 某些资产可能卖不出去，预留 2-3 周时间 — 延迟风险
4. **集中风险** — 调整后检查单项占比是否过高 — 降低系统性风险
5. **通胀侵蚀** — 定期复查配置，年度调整一次 — 长期收益保护

## 再平衡频率建议

| 情况         | 建议频率 | 触发条件        |
| ------------ | -------- | --------------- |
| 市场平稳     | 1 次/年  | 年初或特定日期  |
| 市场波动中等 | 2 次/年  | 某资产偏离 >5%  |
| 市场高波动   | 按需     | 某资产偏离 >10% |
