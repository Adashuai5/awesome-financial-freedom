# Input-Output Mapping (输入-输出映射表)

> 在用户文档中，用表格形式明确说明每个输入字段对应什么系统输出的编写模式。

## Problem Context

当表单/工具的输入字段较多时，用户容易不清楚"填了这些数会得到什么"。
仅靠文字描述各字段或各输出结果，无法建立字段→输出的对应关系。

## Solution

使用两列表格：左列列出相关输入字段组合，右列说明对应的系统输出。
示例（本项目 FIRE 表单）：

| 你的输入 | 系统输出 |
|---|---|
| 当前存款 + 年收入 + 年支出 + 年储蓄 | **FIRE 目标金额** 和 **预计达成年限** |
| 预期回报率、安全提取率 | FIRE 计算参数（复利增长、提取数学） |
| 资产配置（如 60% 股票 / 40% 债券） | **再平衡建议** 和风险说明 |

## Trade-offs

- **优点**：用户一目了然知道"填什么→得到什么"，减少疑惑
- **缺点**：表单字段变更时需要同步更新表格，维护成本略高
- **适用条件**：字段 ≥ 4 个，且输出与字段有明确的分组对应关系时
