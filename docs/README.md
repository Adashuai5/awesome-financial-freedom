# Documentation

This directory contains the main project documentation and tutorial resources for Awesome Financial Freedom.

## What’s inside

- `tutorials/getting-started.md` — step-by-step onboarding for new users
- `book-summaries/` — curated notes and summaries from key financial books
- `../prompts/system-prompt.md` — AI Skill prompt template for Claude and ChatGPT

## Why this matters

This documentation section is the first place contributors and users should visit when they want to:

- learn how to use the repo
- understand the knowledge structure
- review the AI skill prompt flow
- explore book summary resources
